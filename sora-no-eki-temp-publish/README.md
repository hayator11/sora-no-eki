# おのくんの実家 空の駅 公式ホームページ

静的HTML/CSS/JSで構成した公開前たたき台です。

## 公開前に差し替える情報

* `site.config.js` の `siteUrl`
* 正式な独自ドメイン
* 詳細住所、電話番号、営業時間、定休日、駐車場
* Googleビジネスプロフィールの正式URL
* 実際の外観、店内、駅からの道順写真
* 問い合わせフォームURLまたはメールアドレス
* Google Analytics、Google Search Consoleのタグ

## 公式リンク

* おのくん公式サイト: https://onokun.com/
* おのくん公式SHOP: https://onokun.shop.socialimagine.com/

## 確認方法

このフォルダでローカルサーバーを起動し、ブラウザで `http://localhost:8000/` を開いて確認します。

## 仮公開

仮公開用の設定として、`netlify.toml`、`vercel.json`、`_headers`、`.nojekyll` を追加しています。
詳しくは `PUBLISH_TEMP.md` を確認してください。
